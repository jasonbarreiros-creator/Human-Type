Quick Start

  • Run python3 typer_server.py in Terminal — leave it open
  • Open HumanTyper.html in Chrome
  • The pill in the top bar turns green when connected
  • Paste text → pick speed → click START
  • Switch to your target app before countdown ends

Ollama CORS Fix

If chat says "Load failed", Ollama needs CORS permission.

macOS:

launchctl setenv OLLAMA_ORIGINS "*" && pkill Ollama; open -a Ollama

Windows: Quit Ollama, then in cmd:

set OLLAMA_ORIGINS=* && ollama serve

Linux:

OLLAMA_ORIGINS="*" ollama serve

FAQ

▸ Server pill stays red?
Run pip3 install pyautogui then python3 typer_server.py. Leave Terminal open.
▸ Nothing typed into my app?
Click inside your target app before the countdown hits zero.
▸ Mac permissions popup?
System Settings → Privacy → Accessibility → enable Terminal.
▸ Ollama chat says "Load failed"?
See the Ollama CORS Fix section on this page. You need to set OLLAMA_ORIGINS and restart Ollama.
▸ Personal Profile?
Complete 5 Analyzer sessions. The profile learns your WPM, timing, and error rate. Toggle it in the Type tab.
